}{% the abstract
    In the game development community C++ is the most widely used language, as it is common belief that speed is of utmost importance. However, notable figures within the community suggest a shift to the functional paradigm. 
    
    This project explores the state of the art within game development and functional programming. The purpose of this exploration is to answer the question of what language is the best. We do so by suggesting the first building blocks for a language evaluation framework called PROFILE. The first part consists of a \ac{CPU} Utilisation benchmark that comes in form of micro- and macrobenchmarks.
    
    Later research in the same field could improve the framework by taking into account language usability, memory utilisation, \ac{GPU} utilisation and so fourth to provide an allround framework.
